## Joining a Domain
To join a domain using Mac in Windows Domain environment follow these steps : 
1. Choose Settings, then choose user & groups
	![[Pasted image 20250704100941.png]]
2. Select the user account you want to join to the domain then click login options
	![[Pasted image 20250704101015.png]]
3. If the lock icon at the bottom-left corner of the page is locked, click it and enter your password when prompted
4. Click the Join button
	![[Pasted image 20250704101103.png]]
5. Enter The name of the domain, you want to join
	![[Pasted image 20250704101126.png]]
6. Enter the name & password of a domain administrator account, then click ok
	![[Pasted image 20250704101150.png]]
7. Close the User& group Window
## Connecting to a share
It means accessing a network share
A network share is a folder, drive or even a printer on one computer that has been configured to be accessible to other computers on the same network
Once you have joined a domain, you can access its network shares via the finder. Just follow these steps
1. Click finder
	![[Pasted image 20250704101353.png]]
2. Choose Go=> Finder
	![[Pasted image 20250704101414.png]]
3. Type the smb path tha leads to the server share you want to connect to 
	To type a smb path follow this syntax : 
	smb://server-name/share-name
4. Click connect
5. Enter a domain username and password, then click Ok
